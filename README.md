# Anonymous Minimal Code Package

This folder is a compact method-oriented code package for anonymous review.

Files:
- `pipeline_core.py`: core algorithm skeleton (baseline, bias-aware, LLM eval)
- `main_flow.py`: end-to-end pipeline order
- `run_experiment.py`: single-file implementation sketch covering the full workflow
- `data/sample_test.jsonl`: sample input data format

Note:
- This package is intentionally minimal and focuses on method clarity.
- Engineering details and full training infrastructure are omitted for anonymized submission.
